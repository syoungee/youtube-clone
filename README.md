### REACT YOUTUBE CLONE CODING

#### 기능 정리

- 전체 video list 보여주기 ✔️
- video list에서 video 이름, 생성한 채널 이름, 올라간 시간 데이터 보여주기
- video 검색 기능
- UI 반응형(사이트가 작아지면 영상 3줄 -> 2줄 -> 1줄로 정렬)
- video 클릭 시 해당 영상으로 이동(router로 구현)
- video 클릭 시 상세 화면 UI구현(관련 영상들도 노출)
- 상세 화면 UI 반응형 구현
